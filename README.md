# Landing Page UI (Tailwind CSS)

A modern and responsive **landing page UI** built using **HTML + Tailwind CSS (CLI)**.
This project showcases a clean hero section with call-to-action buttons, branding, and smooth UI effects.

---

## Features

* Gradient background design
* Clean and modern UI layout
* Built with Tailwind CSS (CLI)
* Hover effects and transitions
* Responsive and centered layout
* Reusable UI structure

---

## Tech Stack

* **HTML5**
* **Tailwind CSS (CLI)**

---
